# Ali R. Salman Retina Website

Static multi-page site modeled after phretina.com. Built with plain HTML/CSS/JS so it is easy to deploy on Netlify.

## Local development

From the project root:

```bash
python3 -m http.server 5173
```

Open `http://localhost:5173` in your browser.
